## Table of contents

## General Info
This API model an account service according to REST guidelines. 
It's able to create, access, find accounts, exchange rates using an external API and to transfer money between them.

## Technologies
Project is created with:
* Java
* IntelliJ IDEA
* Spring Boot
* Spring Web MVC
* Spring Data JPA
* H2
* JUnit 5
* Mockito
* Git
